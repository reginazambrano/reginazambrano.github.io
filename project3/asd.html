AMITTED STUDENT DAYS BOOKLET

As an office assistant at the Recruitment and Communications Office in the University of Notre Dame, I was assigned with the task to create a program booklet for recruitment weekends in 2019.

In order to stay in brand presence with the rest of the deliverables, I was given some documents to use as inspiration for this final piece.

INSPIRATION

These are some of the elements from other recruitment documents that I used to inspire my design. A given was to keep using the same color palette and the same typefaces, therefore the design was up to me. In order to keep it coherent, I decided to use some of the more prominent elements from these other documents.

PROCESS

I first started with one idea, way less developed. It did not have a lot of the same elements that the deliverables had so upon review and feedback, this had to be changed.

FINAL PRODUCT

After many revisions from different departments and feedback, this was the final product. This booklet was distributed to all admitted students coming to Notre Dame in admitted student weekends.
